Easily print Github markdown with gitprint.com

08 March, 2014

Ever tried printing from Github? (What’s all that Github page crap around your content?) Now, gitprint.com makes it easy to print github markdown with one simple URL change.

Simply view any markdown file on Github, then in your URL bar replace the github.com part of the URL with gitprint.com â€” your markdown file will be rendered to a beautiful, printable PDF.

It works for any blob file view (that’s a URL with blob/master in it) as well as root-repo README.md files (there’s still afew more smarts that need to be added to detect other filenames).

Try it for yourself with the jQuery README.md

https://gitprint.com/jquery/jquery/blob/master/README.md
